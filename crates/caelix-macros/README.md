# caelix-macros

Procedural macros for the [Caelix](https://github.com/Ohanronnie/caelix) Rust backend framework, including `#[injectable]`, `#[controller]`, route attributes, guards, and interceptors.

Most applications receive these macros from the [`caelix`](https://crates.io/crates/caelix) facade and do not need this crate directly.
